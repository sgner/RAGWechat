我想了解React Hooks的使用场景，特别是useEffect的清理函数应该怎么用？

A

React useEffect的清理函数主要用于防止内存泄漏和解除订阅等场景。当组件卸载或者依赖项变化导致effect重新执行前，React会调用清理函数。

使用方法是在useEffect中返回一个函数：

useEffect(() => {
  // 设置订阅
  const subscription = someAPI.subscribe();
  
  // 返回清理函数
  return () => {
    // 取消订阅
    subscription.unsubscribe();
  };
}, [someValue]);

常见的使用场景包括:

  • 清除定时器
  • 取消网络请求
  • 取消事件监听器
  • 取消第三方库的订阅
